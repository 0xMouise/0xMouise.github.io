Typora Markdown 编辑快捷键大全（优化补充版）

说明

本文档基于历史对话内容整理，补充了使用场景说明、操作细节及实用技巧，覆盖 Typora 中 Markdown 编辑的核心快捷键（Windows/Linux 与 Mac 版本对应），适用于日常文档编辑、笔记整理等场景，可直接保存为 Markdown 文件或打印使用。

一、文本格式快捷键

功能 Windows/Linux Mac Markdown 语法 补充说明
加粗 Ctrl + B ⌘ + B 加粗内容 选中文本后使用，直接为文本添加加粗格式
斜体 Ctrl + I ⌘ + I 斜体内容 选中文本后使用，支持嵌套（如 粗斜体）
下划线 Ctrl + U ⌘ + U 下划线内容 部分 Markdown 解析器需开启扩展支持
删除线 Alt + Shift + 5 ⌘ + Shift + 5 删除线内容 选中文本后使用，常用于标记作废内容
行内代码 Ctrl + Shift + `` ⌘ + Shift + `` 行内代码 选中文本后使用，自动包裹代码，避免语法冲突
插入超链接 Ctrl + K ⌘ + K 链接文本 可直接粘贴 URL，支持本地文件路径（./xxx.md）
插入图片 Ctrl + Shift + I ⌘ + Shift + I 支持拖拽图片自动生成路径，alt 文本用于加载失败提示
自动补全符号 Tab Tab - 输入 (/[/{ 后按 Tab，自动补全闭合符号 )/]/}

九、使用技巧与注意事项

1. 快捷键冲突解决

  • 若快捷键无效，可能与系统 / 其他软件冲突（如 Ctrl + Shift + K 可能被浏览器占用）：
    1. 打开 Typora → 点击「文件」→「偏好设置」→「快捷键」；
    1. 在搜索框输入目标功能（如 “代码块”），点击右侧 “修改” 自定义快捷键。

2. 手动触发语法补充

部分功能无需快捷键，输入特定符号后按 空格 即可触发：

  • 无序列表：输入 - / * / + + 空格；

  • 有序列表：输入 数字. + 空格（如 1. ）；

  • 块引用：输入 > + 空格；

  • 代码块：输入 + 空格 + 语言名（如python）+ 回车。
